for_zoop_2014: data folder contains
LakeNameMasterand coords.xlsx - info about lake ID, name, region and central latitude, longitude, number of species have.
ZooplanktonTaxa.xlsx - details of zooplankton taxa
and 28 csv files in Zoop Data folder for each lake
and key.xlsx, speciespool_BM.csv files in wrangled_data folder


# cleaning criteria:
- For a lake community, atleast 20 years long timeseries were considered for >= 2 common species (sometimes grouped on genus level) 
  which present atleast 70% of the study period
- June-Sept months from each year were considered to make consistent sampling effort, years not sampled for these 4 months were excluded.
